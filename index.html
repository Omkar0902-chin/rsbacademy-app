Connect to Firebase

One-time setup — data will sync across all phones

This is a one-time setup. After this, all students and admin will share the same live data — attendance, fees, messages — all synced instantly.
1
Create your free Firebase project

Go to console.firebase.google.com → Click "Add project" → Enter a name like "AcademyApp" → Continue → Create project.

2
Add a Web App & get your config

In Firebase console → Project Settings (gear icon) → "Your apps" → Click Web icon (</>) → Register app → Copy the firebaseConfig values below.

3
Enable Firestore Database

In Firebase console → Build → Firestore Database → Create database → Choose "Start in test mode" → Select a region → Enable.

4
Paste your config below
